### 2. CRUD Operations

Implemented CRUD operations using PHP and MySQL.

CRUD stands for:

- **Create** – Add new records to the database.
- **Read** – Retrieve and display existing records.
- **Update** – Modify existing records.
- **Delete** – Remove unwanted records.

The application allows users to efficiently manage records through a dynamic web interface connected to a MySQL database.

#### Key Features

- Add new records
- View stored records
- Update existing records
- Delete records
- Database connectivity
- Dynamic data retrieval
- SQL query execution

---
